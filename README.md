# ritetag-colors
Colorize hashtags


## usage example 
```html
<script src="/dist/ritetag-colors.js"></script>
<script>
window.onload = function(){
	var rite = ritetagcolors;
	rite.CONSUMER_KEY = '';
    	rite.CONSUMER_SECRET = '';
    	rite.OAUTH_TOKEN = '';
    	rite.OAUTH_TOKEN_SECRET = '';
	rite.init();
};
</script>
```
